Run 'SimpleCalculator.exe', type in arithmetic expression and hit 'Enter' key;
Expression tree will be built, two separate 'visitor' operations will run on the tree, and then result of expression will be evaluated and printed on screen.
Hit the 'Enter' key again to end progam.

Program must be run again in order to enter another expression.

"SimpleCalculator.ppt" gives an overview of the project.

This solution was built with "Microsoft Visual Studio Community 2019".
(.NET Framework 4.7.2)
